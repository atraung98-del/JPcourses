# 🍔 Food E-commerce Website

## 📌 Overview
This is a simple Food E-commerce web application where users can explore food items and order online.

## 🚀 Features
- Responsive design using Bootstrap
- Navigation bar with branding
- Food delivery message section
- Clean UI layout

## 🛠️ Tech Stack
- HTML
- CSS
- JavaScript
- Bootstrap
